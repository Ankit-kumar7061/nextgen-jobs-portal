# 🚀 NextGen Jobs Portal

A Django-based multi-city job portal where users can browse job listings by location.

---

## 📌 Project Overview

NextGen Jobs Portal is a web application built using the Django framework.  
It provides job updates for multiple cities such as Hyderabad, Bangalore, Chennai, and Pune.

---

## 🏗️ Architecture Used

This project follows Django’s **MVT Architecture**:

### 🔹 Model
Stores job information like company, title, eligibility, address, etc.

### 🔹 View
Handles business logic and retrieves job data from the database.

### 🔹 Template
Responsible for UI design and displaying job information dynamically.

---
